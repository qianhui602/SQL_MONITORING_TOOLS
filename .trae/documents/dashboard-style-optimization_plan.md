# Dashboard 页面显示样式优化计划

## 一、现状分析

### 当前 Dashboard 页面结构
1. **统计卡片区域** - 9个指标卡片（CPU、内存、连接数、缓存命中率、磁盘使用率、批处理/秒、锁等待、死锁事件、数据库实例）
2. **数据库连接状态概览** - 实例连接状态列表
3. **图表工具栏** - 实例选择、时间范围、刷新间隔、对比模式、自定义图表、排序
4. **图表网格区域** - 6个趋势图表（CPU、内存、连接数、IO、锁、批处理）

### 当前样式问题
- 统计卡片样式较朴素，仅有左侧颜色条区分
- 缺少图标装饰，视觉层次不够丰富
- 卡片悬停交互效果较弱
- 数据库连接状态区域可以更精致
- 整体缺少微动画和渐变效果
- 图表卡片视觉层次可以优化

## 二、优化目标

1. **提升视觉质感** - 增加渐变、阴影、图标等元素
2. **增强交互体验** - 优化悬停效果、增加微动画
3. **优化信息层次** - 强化重要数据的视觉表现
4. **保持设计一致性** - 与现有主题系统兼容
5. **支持深色模式** - 确保在深浅色主题下都有良好表现

## 三、具体优化方案

### 3.1 统计卡片优化

#### 样式改进
- **左侧图标区域**：为每个指标卡片添加对应图标，使用彩色圆形/圆角背景
- **渐变边框**：将左侧单色条改为渐变效果，增加视觉层次
- **卡片背景**：增加轻微的渐变背景或纹理
- **数值样式**：优化数值字体大小、字重，增加数字的视觉冲击力
- **悬停效果**：卡片悬停时轻微上浮（transform: translateY），阴影增强
- **布局优化**：调整内边距和间距，使内容更加舒适

#### 图标映射
- CPU使用率 → CPU/处理器图标
- 内存使用量 → 内存/内存条图标
- 活跃连接 → 连接/网络图标
- 缓存命中率 → 数据库/缓存图标
- 磁盘使用率 → 硬盘/存储图标
- 批处理/秒 → 闪电/速度图标
- 锁等待 → 锁/等待图标
- 死锁事件 → 警告/死锁图标
- 数据库实例 → 服务器/数据库图标

### 3.2 数据库连接状态区域优化

#### 样式改进
- **头部区域**：增加图标和更明显的标题样式
- **状态指示器**：增加呼吸灯/脉动动画效果（在线状态）
- **列表项**：优化悬停背景、增加左侧状态条
- **布局优化**：调整间距，增加信息密度的同时保持可读性
- **状态标签**：优化在线/离线/禁用标签的样式，增加背景色

### 3.3 图表卡片优化

#### 样式改进
- **卡片圆角**：统一圆角大小，增加精致感
- **标题区域**：增加左侧彩色指示条，强化图表分类
- **悬停效果**：优化悬停时的阴影和边框变化
- **放大图标**：优化放大按钮的样式和交互

### 3.4 工具栏优化

#### 样式改进
- **按钮样式**：优化时间范围按钮的选中和悬停效果
- **选择框**：统一下拉框样式，增加焦点状态
- **开关控件**：优化对比模式开关的视觉效果
- **布局**：调整工具栏内间距和元素间距

### 3.5 整体细节优化

- 统一各区域的圆角、阴影、间距规范
- 增加过渡动画（transition）提升流畅感
- 优化深色模式下的颜色对比度
- 确保响应式布局在不同屏幕尺寸下的表现

## 五、实施步骤

1. **统计卡片重构**
   - 添加图标 SVG 元素
   - 重构卡片 HTML 结构（增加图标区域）
   - 编写新的卡片样式（渐变、悬停、动画）

2. **数据库连接状态区域优化**
   - 优化头部样式
   - 增加状态指示器动画
   - 优化列表项样式

3. **图表卡片样式优化**
   - 优化卡片容器样式
   - 优化标题区域样式
   - 优化悬停效果

4. **工具栏样式优化**
   - 优化按钮和选择框样式
   - 优化开关控件样式

5. **整体细节调优**
   - 统一间距、圆角、阴影规范
   - 深色模式适配
   - 响应式布局检查

## 六、注意事项

1. **保持功能不变**：样式优化不改变任何功能逻辑
2. **性能考虑**：动画效果使用 CSS transform 和 opacity，确保不影响性能
3. **主题兼容**：所有颜色优先使用 CSS 变量，确保深浅色主题正常切换
4. **向后兼容**：不破坏现有拖拽排序、图表交互等功能
5. **渐进增强**：新样式在不支持的浏览器中优雅降级

## 七、风险与应对

| 风险 | 应对措施 |
|------|----------|
| 样式调整影响原有布局 | 使用 CSS 变量和相对单位，逐步调整并测试 |
| 深色模式适配问题 | 每部分样式完成后立即切换深色模式测试 |
| 动画性能问题 | 使用 transform/opacity 动画，避免触发重排 |
| 响应式布局错乱 | 每完成一部分测试不同屏幕尺寸 |
