# SQL 监控平台 - 用户手册

## 1. 系统概述

SQL 监控平台是一个用于监控 MS SQL Server 性能的 Web 应用，提供实时监控、告警通知、历史分析等功能。

### 1.1 系统要求
- 现代浏览器（Chrome 90+、Firefox 88+、Edge 90+、Safari 14+）
- 屏幕分辨率：1280x720 及以上

### 1.2 访问方式
- 浏览器访问：`http://<服务器地址>:3000`
- 默认账号：`Admin` / `Chuz0001`

> 首次登录后请立即修改默认密码。

---

## 2. 登录与认证

### 2.1 登录系统
1. 打开浏览器，访问系统地址
2. 输入用户名和密码
3. 点击"登 录"按钮

### 2.2 找回密码
1. 在登录页面点击"忘记密码？"
2. 输入注册时使用的邮箱地址
3. 查收邮件，获取 6 位数字验证码
4. 输入验证码和新密码
5. 点击"重置密码"

### 2.3 修改密码
1. 点击右上角用户头像
2. 选择"个人设置"
3. 在"修改密码"区域输入当前密码和新密码
4. 点击"保存修改"

## 3. 仪表盘 (总览)

仪表盘是系统的首页，提供关键性能指标的概览。

### 3.1 统计卡片
仪表盘顶部显示 9 个统计卡片：
- **数据库实例**：显示在线/离线实例数
- **CPU 使用率**：当前 SQL Server CPU 使用率
- **内存使用量**：SQL Server 使用的内存量
- **缓存命中率**：缓冲池缓存命中率
- **活跃连接**：当前活跃的数据库连接数
- **磁盘使用率**：磁盘空间使用百分比
- **批处理/秒**：每秒批处理请求数
- **锁等待**：当前锁等待数量
- **死锁事件**：累计死锁事件数

#### 自定义统计卡片
1. 点击工具栏的"自定义"按钮
2. 在"统计卡片"区域勾选/取消勾选要显示的卡片
3. 设置自动保存，刷新后保持

#### 拖拽排序
- 将鼠标悬停在卡片上，拖拽右上角的拖拽手柄可调整卡片顺序

### 3.2 数据库连接状态
显示所有监控实例的实时连接状态：
- **在线**（绿色）：实例连接正常
- **离线**（红色）：实例连接断开
- **已禁用**（灰色）：实例已被禁用

### 3.3 性能趋势图表
图表区域显示 6 个趋势图表：
- CPU 使用率趋势
- 内存使用趋势
- 连接数趋势
- IO 延迟趋势
- 锁等待趋势
- 批处理请求趋势

#### 图表操作
- **选择时间范围**：最近 1 小时 / 6 小时 / 24 小时 / 7 天
- **选择实例**：下拉框选择要查看的实例
- **刷新间隔**：设置自动刷新间隔（5/10/30/60 秒或关闭）
- **对比模式**：开启后可与昨天/上周/上月同时对比
- **查看详情**：点击图表可放大查看

#### 自定义图表
1. 点击工具栏的"自定义"按钮
2. 在"图表"区域勾选/取消勾选要显示的图表
3. 点击"排序"按钮可调整图表顺序

## 5. 死锁监控

自动捕获和分析 SQL Server 死锁事件。

### 5.1 查看死锁列表
1. 点击左侧菜单"死锁监控"
2. 查看死锁事件列表，包括时间、受害者会话等

### 5.2 查看死锁详情
1. 在列表中点击某条死锁记录
2. 查看死锁 XML、涉及的 SQL 语句、涉及的对象

### 5.3 AI 分析
1. 在死锁详情页点击"AI 分析"按钮
2. 系统将使用 DeepSeek AI 分析死锁原因
3. 查看优化建议

## 6. 告警管理

### 6.1 查看告警
1. 点击左侧菜单"告警管理"
2. 查看告警列表，包括类型、严重级别、触发时间
3. 支持按严重级别、时间范围筛选

### 6.2 确认告警
1. 在告警列表中点击"确认"按钮
2. 告警状态变为已确认

### 6.3 通知提醒
- 系统支持声音提醒，新告警触发时播放提示音
- 可在顶部栏点击铃铛图标查看通知
- 点击"静音"图标可开启/关闭声音

---

## 7. 告警规则

### 7.1 查看规则
1. 点击左侧菜单"告警规则"
2. 查看所有告警规则列表

### 7.2 创建规则
1. 点击"添加规则"按钮
2. 填写规则信息：
   - 规则名称
   - 指标分类和指标名称
   - 比较操作符（大于、小于、等于等）
   - 阈值
   - 严重级别（低、中、高、严重）
3. 可选：设置静默时段
4. 点击"保存"

### 7.3 启用/禁用规则
- 在规则列表中点击开关按钮启用或禁用规则

## 8. 实例管理

管理 SQL Server 监控实例。

### 8.1 查看实例
1. 点击左侧菜单"实例管理"
2. 查看所有监控实例列表
3. 查看每个实例的连接状态（在线/离线）

### 8.2 添加实例
1. 点击"添加实例"按钮
2. 填写连接信息：
   - 实例名称
   - 主机地址
   - 端口（默认 1433）
   - 用户名
   - 密码
   - 默认数据库
3. 点击"测试连接"验证配置
4. 点击"保存"

### 8.3 编辑实例
1. 在实例列表中点击"编辑"按钮
2. 修改配置信息
3. 点击"保存"

### 8.4 禁用/启用实例
- 在实例列表中点击开关按钮禁用或启用实例

## 19. 常见问题

### Q: 无法登录系统？
A: 请检查：
1. 用户名和密码是否正确
2. 账号是否被禁用
3. 网络连接是否正常

### Q: 实例显示离线？
A: 请检查：
1. SQL Server 服务是否正常运行
2. 网络连接是否正常
3. 防火墙是否开放 1433 端口
4. 用户名和密码是否正确

### Q: 告警没有触发？
A: 请检查：
1. 告警规则是否启用
2. 阈值设置是否合理
3. 是否处于静默时段
4. 是否处于冷却期

### Q: 如何修改采集间隔？
A: 
1. 以管理员身份登录
2. 进入"系统设置"
3. 修改采集间隔值
4. 保存设置
